# Er
## Translations
- **en: he**
## Type
- _**Substantiv**_
- _**Personalpronomen**_
## Pronunciation
- _**eːɐ̯**_
## Meaning
- **[1] eine männliche Person, ein männliches Tier, ein männlicher Gegenstand oder ein männliches Abstraktum**
- **[1] veraltet, kein Plural: Anrede an eine männliche Person**
