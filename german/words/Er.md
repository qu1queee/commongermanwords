# Er 
## Type 
- _**Substantiv**_ 
- _**Personalpronomen**_ 
## Pronunciation 
- _**eːɐ̯**_ 
- _**eːɐ̯**_ 
## Meaning 
- **eine [[männlich]]e [[Person]], ein männliches [[Tier]], ein männlicher [[Gegenstand]] oder ein männliches [[Abstraktum]]** 
- **''veraltet, kein Plural:'' Anrede an eine männliche Person**
