# Präsident
## Translations
- es: presidente
- en: president, chairman
## Type
- _Substantiv_
## Pronunciation
- **_pʁɛziˈdɛnt_**
## Meaning
### Substantiv
- [1] Staatsoberhaupt einer Republik
- [2] Vorsitzender eines Gremiums, einer Organisation oder ihrer Einrichtungen
