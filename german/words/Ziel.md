# Ziel
## Translations
- es: fin, meta
- en: aim, goal
## Type
- _Substantiv_
## Pronunciation
- **_[t͡siːl](https://commons.wikimedia.org/wiki/File:De-Ziel.ogg)_**
## Meaning
### Substantiv
- [1] Ausrichtung oder Endpunkt einer Bestrebung
- [2] Objekt, das von einem Geschoss getroffen werden soll
