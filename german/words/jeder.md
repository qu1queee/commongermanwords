# jeder
## Translations
- es: cada, todo
- en: each, every
## Type
- _Indefinitpronomen_
## Pronunciation
- **_ˈjeːdɐ_**
## Meaning
### Indefinitpronomen
- [1] ein Einzelner (eine Einzelne, ein Einzelnes) zusammen mit allen anderen einer Gruppe
- [2] ein Einzelner (eine Einzelne, ein Einzelnes) einer Gruppe je für sich
